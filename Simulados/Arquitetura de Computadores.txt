QUESTAO: Qual é a menor e mais elementar unidade que os computadores utilizam para trabalhar?
A: Kilobyte
B: Byte
C: Megabyte
D: Bit
GABARITO: D
---
QUESTAO: O que é um Byte?
A: A menor unidade de medida em um computador.
B: Uma unidade de entrada e saída.
C: Um conjunto de oito bits.
D: Uma unidade de medida para contagem.
GABARITO: C
---
QUESTAO: O que é a CPU?
A: Uma unidade que interpreta e executa instruções.
B: Uma unidade de armazenamento temporário.
C: Uma unidade de controle de dados.
D: Uma unidade de entrada e saída.
GABARITO: A
---
QUESTAO: Qual a principal função da Unidade de Controle (UC)?
A: Fornecer resultados finais do processamento.
B: Controlar a execução de instruções.
C: Armazenar temporariamente os dados.
D: Realizar operações matemáticas.
GABARITO: B
---
QUESTAO: Qual a diferença entre hardware e software?
A: Hardware é a parte física do computador, enquanto software é a parte lógica.
B: Hardware refere-se à parte lógica do computador, enquanto software refere-se à parte física.
C: Hardware e software são termos intercambiáveis.
D: Hardware e software são partes do processador.
GABARITO: A
---
QUESTAO: Qual é a função do contador de programa (PC)?
A: Controlar a execução de instruções.
B: Controlar a transferência de dados entre a CPU e a memória.
C: Armazenar o endereço da próxima instrução a ser buscada.
D: Armazenar temporariamente os operandos das operações.
GABARITO: C
---

D: Uma máquina utilizada apenas para executar tarefas repetitivas.
GABARITO: C
